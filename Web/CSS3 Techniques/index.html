DS

Drop Shadow

A drop shadow can give depth and a sense of layering to the elements on your page

RC

Rounded Corners

Rounded corners on elements can give a 'softer' look to your page

IF

Image Filters

Filters can be applied to an image to alter colour, brightness, add blur, etc.

2D

2D Transforms

Elements can be transformed in 2D (shifted, scaled, rotated, etc.)

WF

Web Fonts

Instead of relying on the fonts installed on the user's computer, you can load fonts from the Web

CO

Columns

It's very easy to display paragraphs of text in columns, making them easier to read

3D

3D Transforms

Elements can be transformed in 3D (shifted, scaled, rotated, etc.)

GF

Gradient Fills

The background of an element can be filled with gradients

TR

Transparency

The background of any element can use a semi-transparent fill

FB

Fixed Background

The background the web page can be fixed in place so content scrolls past it

FP

Fixed Position

Any element on the page can be fixed in place, remaining visible when scrolling

TS

Text Shadow

A drop-shadow can be applied to any text to 'lift' it off the page

MQ

Media Queries

CSS rules can be switched on/off based on window size...
